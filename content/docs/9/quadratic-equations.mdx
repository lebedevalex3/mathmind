# Квадратные уравнения

## Теоретический блок

Квадратное уравнение — это уравнение вида $ax^2 + bx + c = 0$, где $a$, $b$ и $c$ — некоторые числа, причем $a \neq 0$.

### Формула дискриминанта

Для решения квадратного уравнения сначала вычисляют дискриминант по формуле:

$D = b^2 - 4ac$

В зависимости от значения дискриминанта определяют количество корней уравнения:
- Если $D > 0$, то уравнение имеет два различных действительных корня
- Если $D = 0$, то уравнение имеет один действительный корень (два совпадающих корня)
- Если $D < 0$, то уравнение не имеет действительных корней

### Формула корней квадратного уравнения

Если $D \geq 0$, то корни квадратного уравнения вычисляются по формуле:

$x_{1,2} = \frac{-b \pm \sqrt{D}}{2a}$

Где:
- $x_1 = \frac{-b + \sqrt{D}}{2a}$
- $x_2 = \frac{-b - \sqrt{D}}{2a}$

## Пример с решением

**Задача:** Решите квадратное уравнение $2x^2 - 5x - 3 = 0$

**Решение:**

1. Определим коэффициенты уравнения:
   - $a = 2$
   - $b = -5$
   - $c = -3$

2. Вычислим дискриминант:
   $D = b^2 - 4ac = (-5)^2 - 4 \cdot 2 \cdot (-3) = 25 + 24 = 49$

3. Так как $D > 0$, уравнение имеет два различных действительных корня.

4. Найдем корни по формуле:
   $x_{1,2} = \frac{-b \pm \sqrt{D}}{2a} = \frac{5 \pm \sqrt{49}}{2 \cdot 2} = \frac{5 \pm 7}{4}$

5. Вычислим значения корней:
   $x_1 = \frac{5 + 7}{4} = \frac{12}{4} = 3$
   $x_2 = \frac{5 - 7}{4} = \frac{-2}{4} = -\frac{1}{2}$

**Ответ:** $x_1 = 3$, $x_2 = -\frac{1}{2}$
